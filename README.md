# Python图书信息管理系统 python45

## 项目概述

本项目为基于Python开发的图书信息管理系统，主要用于管理图书馆的图书信息、读者信息和借阅信息等。系统设计简洁，功能齐全，能够满足小型图书馆日常管理的需求。

## 技术栈

- 后端：Python
- 数据库：SQLite

## 功能模块

- **图书管理**
  - 图书信息录入
  - 图书信息查询
  - 图书信息修改
  - 图书信息删除

- **读者管理**
  - 读者信息录入
  - 读者信息查询
  - 读者信息修改
  - 读者信息删除

- **借阅管理**
  - 借阅记录查询
  - 借阅记录录入
  - 借阅记录修改
  - 借阅记录删除

## 系统角色

- 管理员：负责系统的日常管理和操作。

## 运行环境

- Python环境（推荐Python 3.x版本）
- SQLite数据库

## 部署步骤

1. 确保Python环境和SQLite数据库已正确安装。
2. 克隆或下载项目源码。
3. 按照以下目录结构部署项目文件。

## 目录结构

```
图书信息管理系统
├── src
│   ├── main.py            # 程序入口
│   ├── models.py          # 数据模型
│   ├── views.py           # 视图层逻辑
│   ├── controllers.py     # 控制器逻辑
│   └── utils.py           # 辅助功能模块
└── database
    └── library.db         # SQLite数据库文件
```
